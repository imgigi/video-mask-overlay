# VideoMaskOverlay

一个 macOS 菜单栏遮罩工具：在屏幕上覆盖一块可调范围的半透明颜色，但鼠标点击会穿透到下面的视频播放器。

## 运行

```bash
swift run
```

## Windows 版本

Windows 版本在 [WindowsMaskOverlay](./WindowsMaskOverlay)。它可以打包成自包含 zip，用户解压后双击 `WindowsMaskOverlay.exe` 即可使用。发布流程见 [WindowsMaskOverlay/README.md](./WindowsMaskOverlay/README.md)。

运行后菜单栏会出现一个遮罩图标。 从菜单里可以：

- 启用或关闭遮罩
- 选择白色或黑色遮罩，也可以直接在菜单里输入 Hex 颜色
- 调整透明度，80% 到 100% 区间在滑杆上有更精细的控制
- 直接在菜单里输入透明度百分比
- 选择一张图片作为遮罩，并调整图片透明度
- 调整显示层级：普通、置顶、强力置顶
- 选择目标窗口，只遮住某一个窗口
- 刷新窗口列表

## 使用方式

先运行工具，再打开视频软件。遮罩窗口默认覆盖屏幕中间一块区域，但不会接收鼠标事件，所以仍然可以点击、拖动、暂停或快进下面的视频播放器。

如果只想遮一个窗口，在菜单栏图标里打开“目标窗口”，选择对应的视频软件窗口。选中后遮罩会自动使用该窗口的真实边界，刚好覆盖整个窗口，并跟随窗口移动和缩放。

默认遮罩颜色是白色，默认透明度是 90%。颜色菜单提供白色、黑色两个快捷项；其他颜色可以在菜单里的 Hex 输入框填写，例如 `#FFFFFF`，也支持不带 `#` 的 `FFFFFF`。透明度也可以直接在菜单输入框里填写 `0` 到 `100`。

“图片遮罩”可以选择一张本地图片作为遮罩，不会切换当前选择的目标窗口。默认状态下，图片左侧对齐目标窗口左侧，高度等比例适配目标窗口高度，宽度按图片原始比例展开，不会拉伸变形。图片和目标窗口重叠的部分使用当前透明度；图片超出目标窗口的部分保持不透明。

如果要退出图片遮罩，直接点击白色、黑色或输入 Hex 颜色即可恢复颜色遮罩。

选择目标窗口时会自动切到“普通”层级。点击或操作目标窗口时，遮罩会自动补回到目标窗口上方；如果最前面的是其他窗口，遮罩不会抢最前，所以其他窗口移到目标窗口上方时会盖住遮罩。如果普通层级盖不住目标视频窗口，可以把“显示层级”调到“置顶”。

“强力置顶”会使用更高的系统窗口层级，并在切换桌面 Space 后主动恢复显示，适合需要遮罩跨桌面持续可见的场景。开启后遮罩会更强地盖住其他窗口。

目标窗口短暂从系统窗口列表里读不到时，遮罩会先保持原样；只有连续约 1 秒都找不到目标窗口，才会认为窗口已经关闭或最小化并隐藏遮罩。

切换桌面 Space 后，工具会在切换动画结束前后自动恢复遮罩，不需要再点击目标窗口。

某些系统版本第一次读取窗口列表时，可能需要在“系统设置 > 隐私与安全性”里允许屏幕录制权限。
